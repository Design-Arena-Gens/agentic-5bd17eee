# ایجنت مدیریت فایل اکسل

این پروژه یک اپلیکیشن Next.js است که به شما اجازه می‌دهد فایل اکسل را بارگذاری کرده، ستون‌های دلخواه را انتخاب یا نام‌گذاری مجدد کنید و خروجی جدیدی به‌صورت فایل اکسل دریافت نمایید.

## پیش‌نیازها

- Node.js نسخه ۱۸ یا بالاتر
- npm یا yarn

## نصب و اجرا

```bash
npm install
npm run dev
```

سپس در مرورگر به آدرس `http://localhost:3000` مراجعه کنید.

## اسکریپت‌ها

- `npm run dev` اجرای توسعه
- `npm run build` ساخت نسخه تولید
- `npm run start` اجرای نسخه تولید
- `npm run lint` اجرای ESLint

## امکانات

- بارگذاری فایل اکسل و انتخاب شیت موردنظر
- جست‌وجو در میان ردیف‌ها
- فعال/غیرفعال کردن ستون‌ها برای خروجی
- تغییر نام ستون‌ها پیش از خروجی گرفتن
- دانلود مستقیم خروجی به‌صورت فایل اکسل جدید

## ساختار پوشه‌ها

```
app/           صفحات و Layout اصلی
components/    کامپوننت‌ها
public/        فایل‌های استاتیک
```
